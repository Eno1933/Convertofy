# Convertofy - PNG/JPG to PDF Converter

Convertofy adalah aplikasi web berbasis HTML, CSS, dan JavaScript untuk mengonversi gambar PNG dan JPG ke format PDF.

## Fitur

- Konversi gambar PNG dan JPG ke PDF
- Dukung multiple file sekaligus
- Drag & drop file untuk upload
- Pratinjau gambar sebelum konversi
- Konversi dilakukan di browser (tanpa server)
- Responsif untuk semua perangkat

## Cara Menggunakan

1. Buka aplikasi Convertofy di browser
2. Upload gambar PNG/JPG dengan:
   - Klik area upload untuk memilih file, atau
   - Drag & drop file ke area upload
3. Lihat pratinjau gambar yang diupload
4. Klik tombol "Konversi ke PDF"
5. Tunggu proses konversi selesai
6. File PDF akan otomatis terunduh

## Teknologi yang Digunakan

- HTML5
- CSS3 dengan Bootstrap 5
- JavaScript (ES6+)
- jsPDF library untuk konversi ke PDF

## Cara Menjalankan Aplikasi

1. Unduh semua file proyek
2. Buka file `index.html` di browser web
3. Aplikasi siap digunakan

## Catatan

- Semua proses konversi dilakukan di browser Anda
- Tidak ada data yang dikirim ke server
- Aplikasi ini tidak menyimpan file Anda

## Lisensi

Proyek ini tersedia untuk penggunaan pribadi dan komersial.
